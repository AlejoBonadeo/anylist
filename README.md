# Dev

1. Clone project
2. Node modules: ```yarn```
3. Copy ```.env.template``` and rename to ```.env```
4. Postgres: ```docker-compose up -d```
5. Server: ```yarn start:dev```
6. ```localhost:3000/graphql```
